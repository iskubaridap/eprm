Subject: Interagency Security Committee (ISC) Assessments in EPRM

Date: Wednesday, 9 April

Time: 1100 EST

Purpose: How to complete an ISC assessment using EPRM. The ISC module is very similar to the DoD ATFP/IDRMP module, but it has the necessary modifications to make it ISC compliant, which includes a separate set of benchmarks, a Facility Security Level (FSL) calculation, and DHS baseline threat preload.

How do I attend? Email EPRMhelp@hii-tsd.com for an invite to the webinar
